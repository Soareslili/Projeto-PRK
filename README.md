# Aplicativo de Reserva de Vagas de Estacionamento

Uma aplicação web para reservar vagas de estacionamento, desenvolvida com React, TypeScript e Vite.

## Funcionalidades

- Visualizar vagas de estacionamento disponíveis
- Reservar vagas através de uma interface intuitiva
- Diálogos para confirmação de reservas
- Design responsivo e moderno

## Tecnologias Utilizadas

- **React**: Biblioteca para construção da interface
- **TypeScript**: Para tipagem estática
- **Vite**: Ferramenta de build rápida
- **Tailwind CSS**: Para estilização
- **React Query**: Para gerenciamento de estado e requisições
- **Shadcn/ui**: Componentes de UI reutilizáveis

## Instalação

1. Clone o repositório:
   ```
   git clone <url-do-repositorio>
   cd prk
   ```

2. Instale as dependências:
   ```
   npm install
   ```

3. Execute o servidor de desenvolvimento:
   ```
   npm run dev
   ```

4. Abra o navegador em `http://localhost:5173`

## Uso

- Navegue pela aplicação usando a barra de navegação
- Visualize as vagas disponíveis na página principal
- Clique em uma vaga para abrir o diálogo de reserva
- Preencha os detalhes e confirme a reserva
